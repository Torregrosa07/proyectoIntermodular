Visual Studio Code

Antes de proceder a al tutorial de la instalación del programa es necesario saber un poco de su historia. En Abril del 2015 Microsoft lanzó Visual Studio Code como un editor de código abierto, es decir, OpenSource. Bajo la licencia MIT. La historia de VSC inició con la idea de Microsoft de ofrecer a la comunidad de aquellos programadores y desarrolladores que no necesitaban toda la complejidad de un entorno de desarrollo, sin embargo, Microsoft ya había desarrollado un IDE como Visual Studio que es un IDE bastante completo principalmente para el desarrollo de aplicaciones, pero Visual Studio es más complejo y pesado, cosa que Microsoft no quería para Visual Studio Code, sino que este fuese una herramienta ligera para proyectos pequeños.

Instalación de VSCode

Como instalar VSCode para Windows:

1- Principalmente debemos ir a la página oficial de Visual Studio Code: https://code.visualstudio.com/ y hacer clic en "Download for Windows". Y se procederá a descargar el archivo de instalación.

2- Una vez descargado el archivo hacemos doble clic en el “VSCodeSetup-x64-x.x.x.exe” para comenzar la instalación. El asistente de instalación te guiará mientras estás en el proceso. Se puede elegir la carpeta donde se instalará, crear accesos directos en el escritorio y agregar Visual Studio Code al PATH para abrirlo desde la línea de comandos.

3- Después de que terminemos la instalación, hacemos clic en "Finish" para completar el proceso.

4- Si no lo hemos abierto al final, iremos al menú de inicio y buscaremos "Visual Studio Code" o hacer clic en el acceso directo del escritorio para abrirlo.

5- En cuanto a la configuración, podemos cambiar el idioma, instalar extensiones para diferentes lenguajes de programación y personalizar nuestro entorno.

Como instalar VSCode para Linux

Para instalar VSCode en Linux debemos identificar la distribución que estemos usando ya que el proceso cambia un poco, en este caso para las que son basadas en Debian y Ubuntu y lo haremos utilizando el repositorio de Microsoft.

1- Abriremos una terminal y pondremos la clave del repositorio de Microsoft: 

sudo sh -c 'echo "deb [arch=amd64 signed-by=/usr/share/keyrings/microsoft-archive-keyring.gpg] https://packages.microsoft.com/repos/vscode stable main" > /etc/apt/sources.list.d/vscode.list'

2- Actualizaremos la lista de paquetes:

sudo apt update

3- Y finalmente instalaremos Visual Studio Code:

sudo apt install code

Instalación en Fedora

Iremos a la página oficial de Visual Studio Code: https://code.visualstudio.com/Download y seleccionaremos la versión para Linux, eligiendo el archivo .rpm para Fedora y otras distribuciones basadas en Red Hat.

Una vez descargado el archivo .rpm, abrimos una terminal y navegamos a la carpeta donde se descargó. Usaremos el siguiente comando para instalar el archivo .rpm:

sudo dnf install code-*.rpm

Con esto, la instalación de Visual Studio Code estará completa en Fedora.

Instalación en Arch Linux (Manjaro)

Si estás utilizando Arch Linux o una distribución basada en Arch como Manjaro, se puede instalar Visual Studio Code fácilmente desde el repositorio AUR (Arch User Repository).

Instalar desde AUR usando un ayudante de AUR:

yay -S visual-studio-code-bin

De esta manera se descargará e instalará la versión más reciente de Visual Studio Code.

Instalación mediante Snap (para cualquier distribución de Linux)

Snap es un sistema de paquetes universal para Linux que facilita la instalación de aplicaciones en cualquier distribución. Si prefieres usar Snap, puedes instalar VS Code con un solo comando: 

sudo snap install --classic code

Instalación mediante Flatpak (para cualquier distribución de Linux)

Flatpak es otra opción para instalar aplicaciones en múltiples distribuciones de Linux. Para instalar VS Code con Flatpak, usa el siguiente comando:

flatpak install flathub com.visualstudio.code

Para ejecutar VS Code con Flatpak, usa:

flatpak run com.visualstudio.code

Verificación de la instalación

Para asegurarnos de que Visual Studio Code se instaló correctamente, abrimos una terminal y escribimos el comando:

code

Esto debería abrir Visual Studio Code.

Instalación de Visual Studio Code en MacOS

1. Dirígete al sitio web oficial de Visual Studio Code: https://code.visualstudio.com y haz clic en el botón "Download for Mac". Esto descargará el archivo .zip de la última versión estable.

2. Una vez descargado el archivo .zip, lo descomprimimos.

3. Arrastra y suelta el archivo "Visual Studio Code.app" en la carpeta Aplicaciones de la Mac.

4. Abre la carpeta Aplicaciones y haz doble clic en "Visual Studio Code.app" para abrirlo.

5. Si es la primera vez que abres una aplicación descargada de Internet, macOS podría pedir confirmación para abrirla.

6. Para configurar Visual Studio Code, instala extensiones y personaliza la interfaz según tus necesidades.

Netbeans

Antes de empezar con el tutorial, es útil conocer un poco sobre NetBeans. Este es un IDE de código abierto que nació en 1996 de la mano de la empresa checa XDevelop. En 2000, Sun Microsystems lo compró y lo convirtió en uno de los IDEs más importantes para Java. Después, cuando Oracle adquirió Sun en 2010, NetBeans continuó dentro de su ecosistema. En 2016, Oracle decidió donar el proyecto a la Apache Software Foundation (ASF), donde sigue siendo un proyecto de código abierto y gestionado por la comunidad. Hoy, NetBeans sigue siendo una opción popular para desarrollar en Java, pero también soporta lenguajes como PHP, C++ y HTML5, y es mantenido activamente por Apache.

Instalación de NetBeans

Para Windows

El proceso de instalación es muy sencillo: primero, descarga el archivo .exe desde la página oficial de NetBeans:

https://netbeans.apache.org/download/index.html

Una vez descargado, ejecuta el archivo y sigue las instrucciones del asistente de instalación. NetBeans se instalará en la ubicación que elijas y podrás empezar a usarlo inmediatamente.

Para Linux

En Linux, se puede instalar desde la terminal usando los repositorios oficiales de distribución o descargando el archivo .tar.gz desde la página oficial. Para las distribuciones basadas en Debian, como Ubuntu, puedes usar el siguiente comando:

sudo apt-get install netbeans

O descargar el archivo comprimido y descomprimirlo:

tar -xvzf netbeans.tar.gz

Para macOS

En Mac, basta con descargar el archivo .dmg desde la página oficial y arrastrar la aplicación a la carpeta de Aplicaciones.

IntelliJ IDEA

IntelliJ IDEA es otro popular IDE, desarrollado por JetBrains. Es ampliamente utilizado en la comunidad Java y está basado en la plataforma IntelliJ. Además de su soporte excepcional para Java, IntelliJ IDEA es compatible con muchos otros lenguajes como Kotlin, Groovy, Scala, entre otros.

Instalación de IntelliJ IDEA

Para Windows

Para instalar IntelliJ IDEA en Windows, visita la página oficial de JetBrains e instala el archivo .exe. El asistente de instalación te guiará a través del proceso.

Para Linux

En Linux, puedes instalar IntelliJ IDEA utilizando el archivo .tar.gz desde el sitio web de JetBrains o utilizando la terminal si tu distribución soporta snap:

sudo snap install intellij-idea-community --classic

Para macOS

En macOS, puedes descargar el archivo .dmg desde la página de JetBrains y arrastrar la aplicación a la carpeta de aplicaciones.

Eclipse

Eclipse es otro IDE muy popular, especialmente entre los desarrolladores Java. Fue lanzado en 2001 y desde entonces ha sido una herramienta fundamental para desarrolladores. Eclipse es open-source y tiene una gran cantidad de complementos que permiten trabajar con varios lenguajes.

Instalación de Eclipse

Para Windows

Descarga el instalador de Eclipse desde su página oficial y sigue las instrucciones para completar la instalación.

Para Linux

En Linux, Eclipse se puede instalar desde el repositorio de tu distribución. Por ejemplo, en distribuciones basadas en Debian o Ubuntu, puedes usar el siguiente comando:

sudo apt-get install eclipse

Para macOS

En macOS, descarga el archivo .dmg desde la página oficial de Eclipse y arrastra la aplicación a la carpeta de Aplicaciones.

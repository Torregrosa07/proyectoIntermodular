Visual Studio Code

Antes de proceder a al tutorial de la instalación del programa es necesario saber un poco de su historia. En Abril del 2015 Microsoft lanzó Visual Studio Code como un editor de código abierto, es decir, OpenSource. Bajo la licencia MIT. La historia de VSC inició con la idea de Microsoft de ofrecer a la comunidad de aquellos programadores y desarrolladores que no necesitaban toda la complejidad de un entorno de desarrollo, sin embargo, Microsoft ya había desarrollado un IDE como Visual Studio que es un IDE bastante completo principalmente para el desarrollo de aplicaciones, pero Visual Studio es más complejo y pesado, cosa que Microsoft no quería para Visual Studio Code, sino que este fuese una herramienta ligera para proyectos pequeños.

Instalación de VSCode

Como instalar VSCode para Windows:

1- Principalmente debemos ir a la página oficial de Visual Studio Code https://code.visualstudio.com/ y hacer clic en "Download for Windows". Y se procederá a descargar el archivo de instalación.

2- Una vez descargado el archivo hacemos doble clic en el “VSCodeSetup-x64-x.x.x.exe” para comenzar la instalación. El asistente de instalación te guiará mientras estás en el proceso. Se puede elegir la carpeta donde se instalará, crear accesos directos en el escritorio y agregar Visual Studio Code al PATH para abrirlo desde la línea de comandos.

3- Después de que terminemos la instalación, hacemos clic en "Finish" para completar el proceso.

4- Si no lo hemos abierto al final, iremos al menú de inicio y buscaremos "Visual Studio Code" o hacer clic en el acceso directo del escritorio para abrirlo.

5- En cuanto a la configuración, podemos cambiar el idioma, instalar extensiones para diferentes lenguajes de programación y personalizar nuestro entorno.

Como instalar VSCode para Linux

Para instalar VSCode en Linux debemos identificar la distribución que estemos usando ya que el proceso cambia un poco, en este caso para las que son basadas en Debian y Ubuntu y lo haremos utilizando el repositorio de Microsft.

1- Abriremos una terminal y pondremos la clave del repositorio de Microsoft:https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > /usr/share/keyrings/microsoft-archive-keyring.gpg Y agregaremos el repositorio de VIsual Studio Code: sudo sh -c 'echo "deb [arch=amd64 signed-by=/usr/share/keyrings/microsoft-archive-keyring.gpg] https://packages.microsoft.com/repos/vscode stable main" > /etc/apt/sources.list.d/vscode.list'

2- Actualizaremos la lista de paquetes: sudo apt update

3- Y finalmente instalaremos Visual Studio Code sudo apt install code

Instalación en Fedora

Iremos a la página oficial de Visual Studio Code: https://code.visualstudio.com/Download y seleccionaremos la versión para Linux, eligiendo el archivo .rpm para Fedora y otras distribuciones basadas en Red Hat.

Una vez descargado el archivo .rpm, abrimos una terminal y navegamos a la carpeta donde se descargó. Usaremos el siguiente comando para instalar el archivo .rpm:

sudo dnf install code-*.rpm

Con esto, la instalación de Visual Studio Code estará completa en Fedora.

Instalación en Arch Linux (Manjaro)

Si estás utilizando Arch Linux o una distribución basada en Arch como Manjaro, se puede instalar Visual Studio Code fácilmente desde el repositorio AUR (Arch User Repository).

Instalar desde AUR usando un ayudante de AUR:

yay -S visual-studio-code-bin

De esta manera se descargará e instalará la versión más reciente de Visual Studio Code.

Instalación mediante Snap (para cualquier distribución de Linux)

Snap es un sistema de paquetes universal para Linux que facilita la instalación de aplicaciones en cualquier distribución. Si prefieres usar Snap, puedes instalar VS Code con un solo comando:

sudo snap install --classic code

Instalación mediante Flatpak (para cualquier distribución de Linux)

Flatpak es otra opción para instalar aplicaciones en múltiples distribuciones de Linux. Para instalar VS Code con Flatpak, usa el siguiente comando:

flatpak install flathub com.visualstudio.code

Para ejecutar VS Code con Flatpak, usa:

flatpak run com.visualstudio.code

Verificación de la instalación

Para asegurarnos de que Visual Studio Code se instaló correctamente, abrimos una terminal y escribimos el comando:

code

Esto debería abrir Visual Studio Code.

Instalación de Visual Studio Code en MacOS

1. Dirígete al sitio web oficial de Visual Studio Code: https://code.visualstudio.com y haz clic en el botón "Download for Mac". Esto descargará el archivo .zip de la última versión estable.

2. Una vez descargado el archivo .zip, lo descomprimimos.

3. Arrastra y suelta el archivo "Visual Studio Code.app" en la carpeta Aplicaciones de la Mac.

4. Abre la carpeta Aplicaciones y haz doble clic en "Visual Studio Code.app" para abrirlo.

5. Si es la primera vez que abres una aplicación descargada de Internet, macOS podría pedir confirmación para abrirla.

6. Para configurar Visual Studio Code, instala extensiones y personaliza la interfaz según tus necesidades.

Netbeans

Antes de empezar con el tutorial, es útil conocer un poco sobre NetBeans. Este es un IDE de código abierto que nació en 1996 de la mano de la empresa checa XDevelop. En 2000, Sun Microsystems lo compró y lo convirtió en uno de los IDEs más importantes para Java. Después, cuando Oracle adquirió Sun en 2010, NetBeans continuó dentro de su ecosistema. En 2016, Oracle decidió donar el proyecto a la Apache Software Foundation (ASF), donde sigue siendo un proyecto de código abierto y gestionado por la comunidad. Hoy, NetBeans sigue siendo una opción popular para desarrollar en Java, pero también soporta lenguajes como PHP, C++ y HTML5, y es mantenido activamente por Apache.

Instalación de NetBeans

Instalación de NetBeans en Windows

1. Dirígete a la página oficial de NetBeans: https://netbeans.apache.org/.

2. Haz clic en "Download" y selecciona la versión para Windows.

3. Inicia la instalación haciendo doble clic en el archivo descargado (.exe) y sigue los pasos del instalador.

4. Elige las herramientas necesarias (Java, PHP, C++, etc.) y selecciona la ubicación de instalación.

5. Una vez finalizada la instalación, abre NetBeans desde el menú de inicio.

Instalación de NetBeans en Linux

1. Visita la página oficial de NetBeans y descarga el archivo .sh para Linux.

2. Asegúrate de tener instalado el JDK en tu sistema. Si no lo tienes, puedes instalarlo usando:

sudo apt install openjdk-11-jdk

3. Da permisos de ejecución al archivo descargado:

chmod +x netbeans-X.X-bin-linux-x64.sh

4. Ejecuta el instalador:

./netbeans-X.X-bin-linux-x64.sh

5. Sigue las instrucciones del instalador y selecciona la carpeta de instalación.

6. Una vez completada la instalación, abre NetBeans desde el menú de aplicaciones o con el comando:

netbeans

Instalación de NetBeans en MacOS

1. Dirígete a la página oficial de NetBeans: https://netbeans.apache.org/.

2. Descarga el archivo .dmg para macOS y abre el archivo descargado.

3. Arrastra el icono de NetBeans a la carpeta Aplicaciones.

4. Abre NetBeans desde la carpeta Aplicaciones y realiza la configuración inicial.

Intellij IDEA

Es un IDE desarrollado por JetBrains, que fue lanzado en 2001. En sus inicios estuvo originalmente centrado en Java. Esto debido a su inteligencia de código, ya que proporcionaba una autocompletado inteligente y análisis de código en tiempo real. Con el pasar del tiempo, Intellij IDEA expandió su soporte a otros lenguajes como Kotlin, Javascript, Scala, Python, etc. En 2011, JetBrains agregó la Community Edition gratuita, mientras que Ultimate sigue teniendo características adicionales que son de pago. En 2017 se volvió aún más popular con el auge de Kotlin, lenguaje que ha sido adoptado por Google para el desarrollo de Android. Hoy en día es conocido por su soporte avanzado para frameworks, análisis de código en tiempo real y nuevas herramientas modernas. Su enfoque en la constante innovación lo han convertido en uno de los IDES más usados del mundo.

Instalación de IntelliJ IDEA

Instalación de IntelliJ IDEA en Windows

1. Dirígete a la página oficial de IntelliJ IDEA: https://www.jetbrains.com/idea/download/.

2. Descarga el instalador para Windows (.exe) y sigue los pasos del asistente de instalación.

3. Abre IntelliJ IDEA desde el menú de inicio y realiza las configuraciones iniciales como el JDK.

4. Verifica la instalación creando un nuevo proyecto.

Instalación de IntelliJ IDEA en Linux

1. Descarga el archivo .tar.gz desde la página oficial.

2. Extrae el archivo descargado y navega a la carpeta bin:

cd ~/Descargas
tar -xzf ideaIC-*.tar.gz

3. Ejecuta el script de inicio:

cd idea-*/bin
./idea.sh

4. Configura IntelliJ IDEA según tus necesidades y verifica la instalación.

Instalación de IntelliJ IDEA en MacOS

1. Descarga el archivo .dmg desde la página oficial.

2. Arrastra el icono de IntelliJ IDEA a la carpeta Aplicaciones.

3. Abre IntelliJ IDEA desde la carpeta Aplicaciones y realiza la configuración inicial.

Eclipse

Este IDE fue lanzado inicialmente por IBM en 2001 como un proyecto interno. Sin embargo, fué en 2003 cuando el proyecto se hizo de código abierto y ahí fué cuando la Eclipse Foundation fué fundada para apoyar el desarrollo de este. En sus inicios fué originalmente creado para Java, pero ahora soporta más lenguajes como C++, Python, etc. También es conocido por su arquitectura que permite que el usuario añada funcionalidades según vaya necesitando. Eclipse cuenta con un editor de código avanzado, es gratuito, personalizable, también cuenta con una gran comunidad de usuarios y desarrolladores que contribuyen a su mejora.

Instalación de Eclipse

Instalación de Eclipse en Windows

1. Dirígete a la página oficial de Eclipse: https://www.eclipse.org/downloads/.

2. Descarga el archivo .exe y sigue las instrucciones del asistente de instalación.

3. Selecciona Eclipse IDE for Java Developers y espera a que se complete la instalación.

4. Abre Eclipse, selecciona el workspace y asegúrate de que el JDK esté instalado.

Instalación de Eclipse en Linux

1. Asegúrate de tener el JDK instalado. Si no lo tienes, usa:

sudo dnf install java-17-openjdk

2. Descarga el archivo .tar.gz desde la página oficial y descomprímelo.

tar -xvzf eclipse-inst-linux64.tar.gz

3. Ejecuta el instalador:

./eclipse-inst

4. Una vez instalado, abre Eclipse con el comando:

eclipse

Instalación de Eclipse en MacOS

1. Descarga el archivo .dmg desde la página oficial.

2. Abre el archivo .dmg y arrastra el icono de Eclipse a la carpeta Aplicaciones.

3. Abre Eclipse desde la carpeta Aplicaciones y realiza la configuración inicial.
